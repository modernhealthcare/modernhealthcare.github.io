Unordered Lists

Default

      
        <ul>
          <li>uno</li>
          <li>dos</li>
          <li>tres</li>
        </ul>

Example:

  • uno
  • dos
  • tres

Dots

      
        <ul class="dot-list">
          <li>un</li>
          <li>deux</li>
          <li>trois</li>
        </ul>

Example:

  • un
  • deux
  • trois

Ordered Lists

Default

      
        <ol>
          <li>eins</li>
          <li>zwei</li>
          <li>drei</li>
        </ol>

Example:

  1. eins
  2. zwei
  3. drei
